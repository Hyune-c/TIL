# HTTP Method

| Method  | Desc                        | Desc2 |
| ------- | --------------------------- | ----- |
| GET     | 리소스 획득                 |       |
| POST    | 엔티티 전송                 |       |
| PUT     | 파일 전송                   |       |
| HEAD    | 메시지 헤더 취득            |       |
| DELETE  | 파일 삭제                   |       |
| OPTIONS | 제공하고 있는 메소드의 문의 |       |
| TRACE   | 경로 재조사                 |       |
| CONNECT | 프록시에 터널링 요구        |       |
